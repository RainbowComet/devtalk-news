Американська технологічна компанія Block, яку пов’язують із Джеком Дорсі, відкрила у публічний доступ власний AI-інструмент для розробників під назвою Goose.

Проєкт позиціонується як безкоштовна open-source альтернатива таким рішенням, як Claude Code та GitHub Copilot. На тлі зростання цін на AI-інструменти для програмування це робить Goose особливо помітним на ринку developer tooling.

За заявленими можливостями, Goose може працювати з різними великими мовними моделями, включно з Claude, GPT, Gemini, Llama та DeepSeek. Інструмент отримує доступ до кодової бази, може читати й редагувати файли, запускати shell-команди, встановлювати залежності, виконувати код і допомагати з дебагом.

Окремо Block робить акцент на тому, що Goose доступний у кількох форматах — як desktop-застосунок, CLI-інтерфейс і web-версія. Сам проєкт написаний на Rust, що відповідає тренду на легкі та продуктивні developer tools.

Особливо показовим є підхід компанії: спочатку Goose створювався як внутрішній інструмент для інженерів Block, а згодом був відкритий для всієї спільноти. Це продовжує тренд, коли великі технологічні компанії перетворюють внутрішні AI-рішення на open-source продукти.

На тлі швидкого розвитку AI-assisted development Goose може стати помітним гравцем у сегменті coding agents — особливо для тих, хто шукає альтернативу закритим платним платформам.
